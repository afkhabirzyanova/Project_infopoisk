# Project_infopoisk

# Поисковая система по корпусу порошков

## Описание проекта

Проект представляет собой поисковую систему по корпусу текстов (стишков-порошков).

На вход подается текстовый запрос, на выход — наиболее релевантные документы из корпуса.

---

## Данные

Используется корпус порошков (1000 текстов), собранный на основе материалов паблика:
https://github.com/tadgeislamins/poroshki_corpus

---

## Предобработка

Для текстов используется:

- приведение к нижнему регистру
- удаление пунктуации
- токенизация

Дополнительно:

- для BM25 — лемматизация (pymorphy3)
- для Word2Vec — леммы + POS-теги
- для fastText — исходные словоформы (для работы с n-граммами)

Стоп-слова не удаляются, так как тексты короткие и служебные слова могут быть значимыми.

---

## Индексы

### BM25
Классическая модель информационного поиска на основе частот слов.  
Реализована с помощью библиотеки `rank_bm25`.

---

### Word2Vec
Документы представлены как средние векторы слов.  
Используется предобученная модель:

- `word2vec-ruscorpora-300` (gensim downloader)

Сходство — косинусная близость.

---

### fastText
Использует символьные n-граммы, что позволяет работать с несуществующими словами.  

Используется предобученная модель:

- `cc.ru.300.bin` (Facebook fastText)

Добавлено улучшение: бонус за совпадение слов между запросом и документом.

---

## Поиск

На вход подается:
- текст запроса
- тип индекса

На выходе:
- топ документов
- значение сходства
- время выполнения поиска
